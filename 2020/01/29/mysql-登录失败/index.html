mysql 登录失败

 2020-01-29  2020-03-11 a few seconds read (About 46 words)

mysql -u root -p报错

ERROR 2002 (HY000): Can’t connect to local MySQL server through socket ‘/tmp/mysql.sock’ (2)

这是因为mysql服务没有启动
启动命令如下

mysql.server start
or
brew services start mysql
